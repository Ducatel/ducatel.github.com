Générateur de captcha:

  • Devellopé en PHP5
  • Basé sur la librairie graphique GD
  • Simple à utiliser
Télécharger les sources
Exemple d'implémentation

Page d'insertion du Captcha:

				  <?php
				  // A mettre avant toute sortie HTML
				  session_start();
				  ?>
				  //Formulaire du captcha
				  <form method="post" action="traitement.php">

				  // Pour generer le captcha, il suffit de mettre captcha.php dans une balise image
				  <img src="captcha.php" alt="Captcha" id="captcha"
				  onclick="document.images.captcha.src='captcha.php?id='+Math.round(Math.random(0)*1000)+1"
				  title="Cliquer pour changer d'image" style="cursor:pointer;" />
				  <p>
				  <label for="userCode">Entrez le code:</label>
				  <input name="userCode" id="userCode" type="text" />
				  <input type="submit" name="submit" id="submit" value="Valider" />
				  </p>
				  </form>
Page de traitement:
				  <?php
				  session_start();

				  // si l'utilisateur a bien tappe un code
				  if (!empty($_POST['userCode'])){

				  //Convertion du texte est majuscule
				  $userCode = strtoupper($_POST['userCode']);

				  // Comparaison du code
				  if( md5($userCode) == $_SESSION['captcha'] )
				  echo '<h2 class="correct">Correct !</h2>'; // Le code est bon
				  else
				  echo '<h2 class="incorrect">Incorrect recommencez !</h2>'; // Le code est erroné
				  }
				  else
				  echo '<h2>Veuillez entrer le code</h2>'; // ici aucun code detecte

				  ?>


Script de compression ZIP d'arborescence:
  • Devellopé en PHP5
  • Basé sur l'extension ZIP de PHP5
  • Simple à utiliser
Télécharger les sources
Exemple d'implémentation 

				<?php

				// Inclusion de la class d'archivage
				include('ZipBackup.php');

				// Le chemin vers l'arborescence
				$dossier="./dossier";

				//Nom de l'archive
				$archive=date('d-m-Y').".zip";

				// Envoie des header pour le telechargement de l'archove
				header('Content-Type: application/x-zip');
				header("Content-Disposition: attachment; filename=".$archive);

				// Creation de l'objet archive
				$zip=new ZipBackup($archive);

				// Compression du dossier
				$zip->addRecursive($dossier);

				// Fermeture de l'archive
				$zip->close();

				// Envoie de l'archive
				readfile($archive);

				?>
